IMPORTANT NOTES
---------------

1. Use Different Client IDs:
   - Web Client ID: for frontend OAuth flow
   - Android Client ID: auto-configured, just needs SHA-1
   - iOS Client ID: auto-configured, just needs Bundle ID

2. Security:
   - NEVER commit .env file to git
   - Add .env to .gitignore
   - Use different credentials for prod vs dev

3. Common Issues:
   - "Play Services not available": Use real device instead of emulator
   - "Invalid Web Client ID": Check .env file has correct ID
   - "SHA-1 mismatch": Regenerate fingerprint and update OAuth credentials

4. Testing Sequence:
   [1] Email/Password login (basic auth)
   [2] Google OAuth on Android
   [3] Google OAuth on iOS
   [4] Database verification
   [5] Error handling (try wrong password, etc)
